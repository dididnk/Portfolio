Editer la note

Vous pouvez créer une note pour chaque document (Image 1).
Ce texte est indexé et donc faire l'objet de recherches.
Si la note est modifiée, le texte précédent est historisé sous le champ "Note" (Image 2)

Cliquez sur "sauver" pour enregistrer la note saisie.

Le texte de la note apparaît  ensuite sous le résumé du document affiché dans la liste des réponses
du moteur de recherche.
