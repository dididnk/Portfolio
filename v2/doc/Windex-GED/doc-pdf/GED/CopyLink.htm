Copier le lien

 

Ce module permet d'avoir accès aux hyperliens du fichier sélectionné ou du répertoire courant dans lequel il se trouve.

Cela permet, par exemple, de partager le lien dans le but de fournir un accès rapide au fichier.

Cliquez sur pour sélectionner tout le champ, puis copier le lien (Ctrl+C ou cmd+C (MAC) ou clic droit -> copier)
