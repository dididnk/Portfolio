Les propriétés du document

 

Ce module vous permet de modifier les differentes propriétés d'un document :
Etat
 Etat du document.
Titre
 Titre du document.
Un document dont les mots recherchés se trouvent dans le titre sera jugé plus pertinent.
Il sera proposé en priorité dans la liste des réponses du moteur de recherche , si toutefois le tri par pertienne a été demandé.
Résumé
 Résumé du document
Un document dont les mots recherchés se trouvent dans le résumé sera jugé plus pertinent, mais moins que s'il se trouve dans le titre.
Commentaire
 Commentaire du document.
Le commentaire n'est pas indexé par le moteur de recherche plein texte
Cliquez sur "sauver" pour enregistrer les propriétés modifiées.
