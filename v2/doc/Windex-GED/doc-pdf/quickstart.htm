Démarrage rapide

Lancement de Windex Server

Le lancement de Windex Server dépend du type d'installation choisie.

1. Installation en mode "Standard"

  • Si Windex Server est installé en tant que service Windows :
    Allez dans "Outils d'administration" puis "Services" pour démarrer ou arrêter Windex Server.
  • Sinon vous trouverez dans le dossier des raccourcis deux liens vers les programmes
    StartWindexServer et StopWindexServer
    qui respectivement démarrent et arrêtent Windex Server.
    Dans ce cas de figure, vous devez impérativement arrêter Windex Server par la commande StopWindexServer, au risque de corrompre irrémédiablement la base de données de Windex Server.

2. Intégration au serveur de servlets Tomcat

Le démarrage et l'arrêt de Windex Server sont dépendants du démarrage et de l'arrêt de Tomcat.

Accès au module d'administration et de recherche

Les accès dépendent du type d'installation choisie.
Par défaut, l'identifiant de l'administrateur est "administrateur" et son mot de passe est "WindexServer"

Windex Server en mode "Standard"

Module d'administration

http://mon_serveur:8087/WindexServer/manager.htm

Module de recherche

http://mon_serveur:8087/WindexServer/Search.htm

Windex Server dans Tomcat

Module d'administration

http://mon_serveur:8080/WindexServer/manager.htm

Module de recherche

http://mon_serveur:8080/WindexServer/Search.htm

où mon_serveur correspond à l'adresse du serveur sur lequel est installé Windex Server.

Activation de Windex Server

Dés la première utilisation, vous devez impérativement activer le logiciel.
Cliquez sur l'icône et suivez les instructions de validation de votre licence.

Changement du mot de passe de l'administrateur

Il est recommandé de modifier le login et le mot de passe de l'administrateur via l'icône
Les informations seront prises en compte après avoir sauvegardé la configuration en cliquant sur l'icône

Quitter Windex Server

Pour quitter le module d'administration, cliquez sur l'icône

Pour quitter le module de recherche, fermez simplement l'instance de votre navigateur

Pour arrêter Windex Server, suivez la procédure correspondant à votre installation et explicité plus haut dans la section "Lancement de Windex Server"
