Le module de sauvegarde

Ce module permet de sauvegarder et restaurer la base de données des index et des métadonnées.
Les documents se trouvant dans l'entrepôt doivent être sauvegardés séparément.

Paramêtrage de la sauvegarde



Dans le champ "Dossier" indiquez le dossier qui contiendra les sauvegardes.
Le champ "Historique" permet d'indiquer le nombre de sauvegardes que vous souhaitez conserver.

Le champ "Activer" permet de lancer en automatique la sauvegarde à des heures et des jours précis.
Le champ "Horaire" indique l'heure de la sauvegarde.
Les jours choisis doivent être sélectionnés en cochant la case les consernant.

Gestion des sauvegardes



Cette écran permet de lancer la sauvegarde en cliquant sur le bouton .
Il affiche égelement la liste des sauvegardes conservées.
Pour chaque sauvegarde, il est possible de :

  • la supprimer. Bouton
  • ou la restaurer. Bouton
