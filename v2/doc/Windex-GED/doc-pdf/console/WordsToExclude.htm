Liste des mots vides

 

Cette liste énumère les mots sur lesquels une recherche ne serait pas pertinente ou nécessaire.

Cette liste peut être complétée en saisissant un mot dans la zone Ajouter un mot puis en cliquant sur le bouton .

Vous pouvez également utiliser un des dictionnaires prédéfinis proposé dans la liste Ajouter un dictionnaire.
Effacez tous les mots

Pour supprimer un mot, cliquez dessus.

Pour supprimer toutes les mots, cliquez sur le bouton . Vous devrez confirmer la suppression avant qu'elle ne soit effective.

Les mots vides sont des mots non significatifs lors d'une recherche.
Exemple :
ne, vous, il, elle, car, dont, pour, une, un, leur
Tout mot non significatif dans une recherche peut être exclu de l'indexation en plus des mots vides.
Par exemple le mot windex est présent dans toutes les pages de la documentation; je l'exclus de l'indexation.
Il est vivement conseillé d'ajouter à cette liste les opérateurs booléens courament utilisés pour la recherche (ainsi que leurs synonymes) afin d'éviter toute confusion.
