URLs à exclure de l'indexation

 Indiquez dans cette liste la ou les formes que doivent respecter les URLs des documents à ne pas indexer.

Cette fonction permet d'accélérer l'indexation en évitant à Windex Server de parcourir des fichiers inutiles ou non indexables (images, sons, vidéos, scripts CGI, bases de données).

Vous pouvez utiliser des caractères génériques suivants pour indiquer les noms des fichiers à exclure :

    * pour remplacer n'importe quel groupement de lettres.
    ? pour remplacer un seul caractère.
    / est utlisé pour séparer les dossiers.
Exclusion Signification
*.gif
 		N'indexe pas les fichiers portant l'extensiongif.
*/rep/* N'indexe pas le contenu des dossiers rep ni de leur sous-dossiers.
*/??.* N'indexe pas les fichiers ne comportant que deux caractères dans leur nom.
*/*.?? N'indexe pas les fichiers dont l'extension comporte exactement deux caractères.


Saisissez la valeur dans la zone puis cliquez sur le bouton pour ajouter l'élément à la liste.

Pour modifier une URL présente dans la liste, corrigez la valeur, puis cliquez sur le bouton

Pour supprimer une URL, cliquez sur le bouton

Pour supprimer toutes les URLs déjà saisies, cliquez sur le bouton
Vous devrez confirmer la suppression avant qu'elle ne soit effective.
