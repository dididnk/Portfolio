Le compte Administrateur

Par défaut, à l'issu de l'installation l'identifiant et le mot de passe de l'administrateur sont 

Identifiant

administrateur

Mot de passe

WindexServer

Nous vous engageons vivement à les modifier.
Pour cela, saisissez les informations les nouvelles informations et validez.
Si les informations concernant l'identification actuelle de l'administrateur sont correctes, les nouvelles informations seront acceptées.
Le nouvel identifiant et/ou mot de passe seront effectifs dès lors que vous aurez enregistré la configuration.

Le champ : "Liste d'IP de l'administrateur"
Par défaut, cette zone est vide. L'administrateurpeut donc se connecter au module d'administration depuis n'importe quel ordinateur.
Afin de garantir la sécurité, vous pouvez indiquer une liste d'adresse IP à partir desquelles l'administrateur pourra se connecter.
Si plusieurs adresses sont saisies, elles doivent être séparées par un point virgule ou une virgule.

Si vous saisissez une adresse IP fausse, vous ne pourrez plus vous connecter à Windex Server.
Autorisez au moins l'adresse 127.0.0.1 (adresse locale) ou l'adresse IP du serveur sur lequel il est installé.
