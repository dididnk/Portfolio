Guide de démarrage rapide

Pour mettre en oeuvre Windex Server, vous devez suivre les étapes suivantes :

  • Activez impérativement votre licence
  • Indiquez l'URL à indexer
    Choisir "URL à Indexer" dans le menu de gauche,
    puis saisir l'URL correspondant aux documents à indexer.
    Par exemple "http://www.lug.com/" ou "file://localhost/"
  • Sauvegardez votre projet
    Icône disquette dans le menu du haut
  • Si vous avez acquis le module GED, vous devez définir un utilisateur dans le carnet d'adresses.
    Choisir "Carnet d'adresses" dans le menu de gauche
    Créez un Profil utilisateur, puis un utilisateur (un mot de passe doit lui être associé)
  • Lancez l'indexation
