Planification

Les documents doivent être régulièrement indexés. Ce planificateur permet d'indiquer l'heure de démarrage de l'indexation, et ce pour chaque jour.
Le module GED permet, en plus du planificateur, d'indexer un document lors de son ajout au moyen de l'Explorer.

Planification

L'indexation peut être planifiée sur une semaine, à raison de 2 indexations au plus par jour. Les heures sont exprimées de 00:00 à 23h59. Windex Server ne lancera une nouvelle indexation que si la précédente s'est terminée.

L'indexation n'a lieu pour une journée que si vous avez saisi au moins une heure : si vous n'en saisissez aucune pour une journée, aucune indexation n'aura lieu pour cette dernière.

Les modifications de planification ne prendront effet qu'après avoir enregistré la configuration.

Nombre d'indexation simultanées

Ce paramètre indique le nombre de documents que Windex Server peut traiter simultanément lors du processus d'indexation.
