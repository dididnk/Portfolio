Paramétrage SMTP de l'envoi d'Email


Ce module permet de configurer les parmètres SMTP utilisés par Windex Server pour l'envoie des courriels.

Description des champs

Adresse du serveur SMTP Indiquez ici l'adresse du serveur SMTP
Adresse Email de l'expéditeur Indiquez ici l'email de l'expéditeur
Login (option)  Indiquer ici (en option) le login pour se connecter au serveur SMTP
Mot de passe (option)  Indiquer ici (en option) le mot de passe associé au login

Cliquez sur "Sauver" pour sauvegarder les paramètres.

Le bouton "Tester" permet de vérifier les paramètres saisis.


Saisissez une adresse email valide, le sujet du courrier, le message puis cliquez sur Tester.
Si une erreur d'envoi survient Windex Server vous l'indique.
Sinon, il affiche le message "Aucune erreur n'a été détectée."
