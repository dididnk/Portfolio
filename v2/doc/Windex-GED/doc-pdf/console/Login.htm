Connexion

L'administration de Windex Server se fait exclusivement par le protocole HTTP, via l'adresse

http://mon_serveur:8080/WindexServer/manager.htm

où mon_serveur correspond à l'adresse du serveur sur lequel est installé Windex Server. Il est suivit par le numéro du port qui dépend de votre installation, à savoir

  • 8087 lorsqu'on utilise Windex Serveren mode "Standard"
  • 8080 lorsqu'on utilise le serveur de servlets Tomcat.
Vous pouvez également accéder à Windex Server via l'adresse http://mon_serveur:8080/WindexServer/index.htm
Cette page vous propose alors deux liens :
  • l'un vers le module d'administration,
  • l'autre vers le module de recherche ou de "Gestion documentaire" (module GED).

L'accès au module d'administration de Windex Server est contrôlé par un nom d'utilisateur et un mot de passe.

Saisissez ces informations puis cliquez sur le bouton .
Si ces informations sont erronées, vous serez redirigé à nouveau vers l'écran d'identification.

Lors de la première utilisation de Windex Server, l'identifiant et le mot de passe de l'administration sont par défaut :

Identifiant

administrateur

Mot de passe

WindexServer

L'identifiant et le mot de passe peuvent être changé via l'écran d'identification de l'administrateur.
