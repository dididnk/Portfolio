Le tableau de bord rassemble les indicateurs sur l'état de Windex Server.

Le bouton permet de supprimer l'ensemble des index.
Pour retrouver un document, vous devez relancer une indexation afin de générer à nouveau les index.

Le bouton vous dirige vers la page des Url en Erreur qui affiche la liste des problèmes rencontrés lors de l'indexation.
