Recherche : mise en évidence des mots trouvés

Il est possible de mettre en évidence les mots trouvés dans les documents HTML.

La mise en évidence n'est pas possible si la requête utilise des jokers * ou ?.

Vous devez d'abord activer la case à cocher "Mise en évidence des mots trouvés" pour pouvoir choisir parmi les options proposées.

a) Mise en évidence standard

6 options sont disponibles :

  • Clignotement du texte
    Cette option est devenue obsolète et n'est plus prise en compte que par quelques rares navigateurs (en particulier d'anciennes versions de Netscape).
  • Gras
  • Emphase
    Sur beaucoup de plateforme, cette option affiche le texte en italique.
  • Italique
  • Souligné
  • Couleur de fond du texte en jaune

Ces options peuvent être cumulées.

Le code utilisé dans les pages HTML pour générer le ou les effets choisis est affichés dans 4 zones de textes en dessous. Plusieurs cas sont distingués, non seulement pour permettre plus de souplesse, mais également à cause de l'option Navigation.

Vous pouvez à votre convenance, modifier le code proposé.

Il n'y a aucun contrôle de validité du code saisi. Nous vous engageons à faire un test avec un éditeur HTML pour vérifier le code à écrire.

b) Navigation

Cette option permet d'activer la navigation entre les occurrences d'une même page, par l'intermédiaire d'hyperliens situés avant et/ou après l'occurrence.

On distingue 3 cas :

  1. l'occurrence est la première de la page
    Elle sera suivie d'un hyperlien pour atteindre l'occurrence suivante
  2. l'occurrence est la dernière de la page
    Elle sera précédée d'un hyperlien pour atteindre l'occurrence précédente
  3. l'occurrence n'est ni la première ni la dernière de la page.
    Elle sera précédée d'un hyperlien pour atteindre l'occurrence précédente et suivie d'un hyperlien pour atteindre l'occurrence suivante.

Le schéma ci-dessus indique à quel endroit le contenu des différentes zones de code sont insérées :

  1. Avant le premier mot trouvé dans la page
  2. Avant chaque mot trouvé (sauf le premier)
  3. Après chaque mot trouvé (sauf le dernier)
  4. Après le dernier mot trouvé
