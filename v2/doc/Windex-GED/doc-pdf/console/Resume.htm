Le résumé s'arrête au dernier point se trouvant entre la valeur minimale et la valeur maximale.

Champ Résumé

La zone Nom du champ contient :

  • Pour les documents HTML, le nom de la balise META qui sera utilisée comme résumé de la page.
  • Pour les documents PDF, le champ "Subject" correspond au résumé.

Le résumé est constitué :

  • soit du contenu du champ description s'il existe.
  • soit des premiers mots de la page si "oui" est indiqué pour :
    Génère un résumé si le champ "Résumé" est vide

La priorité permet d'influencer le calcul de la pertinence d'une réponse. Se reporter à l'aide sur Indexation des champs.
