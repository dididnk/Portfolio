Pages de type sommaire
Parcourir les liens d'une page sans l'indexer

Une page sommaire est analysée pour en extraire ses hyperliens, mais le contenu même de la page n'est pas indexé : aucune recherche ne pourra être effectuée sur cette page.
De même cette page ne sera donc pas proposée dans la liste des réponses.

Saisissez la valeur dans la zone puis cliquez sur le bouton pour ajouter l'élément à la liste.

Pour modifier une URL présente dans la liste, corrigez la valeur, puis cliquez sur le bouton

Pour supprimer une URL, cliquez sur le bouton

Pour supprimer toutes les URLs déjà saisies, cliquez sur le bouton .
Vous devrez confirmer la suppression avant qu'elle ne soit effective.

Vous pouvez utiliser les caractères génériques suivants pour indiquer le nom d'une page sommaire :

    *  pour remplacer n'importe quel groupement de lettres.
    ? pour remplacer un seul caractère.
    / est utlisé pour séparer les dossiers.

Page de type sommaire Signification
*/index.htm Considère tous les fichiers de nom index.htm ,
quelque soit leur position dans l'arborescence, comme des pages de type sommaire.
*/rep/sommaire.htm Considère tous les fichiers de nom sommaire.htm contenus dans un répertoire
nommé rep, comme des pages de type sommaire.
*/ Considère tous les répertoires comme des pages de type sommaire
(cette écriture n'est pas toujours reconnue).
